Dropil Chain Documentation

Welcome to the Dropil Chain documentation portal! We currently offer informational documentation for Dropil Chain and API documentation for the Dropil hosted public API of Dropil Chain.

Info Docs

The informational documentation on Dropil Chain is geared towards helping you learn about the technology and assist in utilizing it's capabilities. Learn how to do various actions on chain such as setting up a full node, becoming a validator, creating a wallet, sending a transaction, etc.

Public API Docs

The public API documentation is useful for developers that would like to query Dropil Chain endpoints without setting up a full node. We provide a free to use public API that supports all endpoints that a full node supports.
